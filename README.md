# Estate Project

A web application for displaying properties in an estate with a modern UI.

## Features
- Responsive hero section with image grid
- Property cards with hover effects
- Contact form
- About & Team sections
- Portfolio and Logistics sections

## Tech Stack
- HTML, CSS (Poppins font)
- Flexbox & Grid for layouts
- Responsive design with media queries

## How to Run
1. Clone the repo
2. Open `index.html` in a browser
